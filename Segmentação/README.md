# Segmentação de imagens

O projeto utiliza de segmentação de imagens, binarização e rotulagem com flood fill para realizar a contagem e demarcação de grãos de arroz a partir de uma imagem.

## Saída
O programa gera:

- Imagem binarizada  
- Imagem final com os grãos identificados e delimitados  
- Contagem total de componentes detectados

## Como executar
1. Certifique-se de ter Python instalado
2. Instale as dependências:
   ```bash
   pip install numpy opencv-python
3. Execute o programa
   ```bash
   python main.py
