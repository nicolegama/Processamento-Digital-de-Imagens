# Filtro de Média 

Este projeto implementa o **filtro da média (blur)** utilizando três abordagens diferentes, com o objetivo de **comparar desempenho e funcionamento** entre os métodos.

## Objetivo
Implementar e analisar três algoritmos para aplicação do filtro de média:

- Algoritmo **ingênuo**
- Filtro **separável**
- Algoritmo com **imagem integral**

Além disso, é feita uma comparação com a implementação da biblioteca OpenCV.
## Saída
O programa gera:

- Imagem original
- Imagem borrada com cada método (incluindo o do OpenCV) 
- Tempo de execução de cada imagem 

## Como executar
1. Certifique-se de ter Python instalado
2. Certifique-se de ter a imagem na mesma pasta que o "main.py"
3. Instale as dependências:
   ```bash
   pip install numpy opencv-python
4. Execute o programa
   ```bash
   python main.py
